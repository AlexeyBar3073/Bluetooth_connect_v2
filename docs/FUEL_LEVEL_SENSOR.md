# Схема подключения датчика уровня топлива (Toyota RAV4 2004)

## Обзор

Датчик уровня топлива Toyota RAV4 — это **два последовательных поплавка** (основной + дополнительный для компенсации наклона).

| Параметр | Значение |
|----------|----------|
| Основной поплавок (полный → пустой) | ~54 Ом |
| Дополнительный поплавок | ~53 Ом |
| **Суммарное сопротивление** | **~107 Ом** |
| Тип | Реостат (переменное сопротивление) |
| Расположение | Топливный бак (в сборе с бензонасосом) |

---

## Схема подключения

### ⚠️ ПРОБЛЕМА: Влияние на штатный индикатор

Если подключить делитель **напрямую к поплавку**, он создаст **параллельную нагрузку**:

```
Приборная панель (5В) ───[Поплавок 0-107 Ом]─── GND
                               │
                               ├─── Наш делитель (100 Ом) ← ВЛИЯЕТ НА ШТАТНЫЙ!
```

| Состояние | R_поплавка | Наш R | Эквивалент | Ошибка штатного |
|-----------|-----------|-------|------------|-----------------|
| Полный бак | 0 Ом | 100 Ом | 0 Ом | 0% ✅ |
| Полупустой | 54 Ом | 100 Ом | 35 Ом | **-35%** ❌ |
| Пустой бак | 107 Ом | 100 Ом | 52 Ом | **-51%** ❌ |

**Результат:** Штатный индикатель будет завышать показания!

### Решение: Буферный усилитель (TL072 / LM358)

Используем **ОУ с высоким входным сопротивлением** (≥1 ТОм) — не влияет на штатную систему:

```
                    TL072 / LM358
                    ┌──────────┐
    От поплавка ────┤(+) OUT   ├───┬─── GPIO34 (ADC)
    (через развязку)│          │   │
                    │(-)   V+  │  [100 Ом]
                    │          │   │
    GND ────────────┤     V-   ├───┘
                    └──────────┘
                          │
    +3.3V ────────────────┘
    -3.3V или GND ────────┘
```

**Характеристики TL072:**
- Входное сопротивление: **10¹² Ом** (1 ТОм!)
- Ток входа: **1 пА** (практически 0)
- Влияние на штатную систему: **0%** ✅
- Точность: **±0.05%** (как без ОУ)

### ⚠️ КРИТИЧНО: Использовать 3.3В (НЕ 5В!)

Если использовать **5В**, при полном баке (0 Ом поплавка):
```
Vadc = 5.0 × 100 / (0 + 100) = 5.0В  ❌ GPIO34 СГОРИТ! (макс 3.3В)
```

С **3.3В** питанием:
```
Vadc = 3.3 × 100 / (0 + 100) = 3.3В  ✅ Безопасно!
```

### Расчёт напряжения

**Формула делителя напряжения:**

```
Vadc = 3.3В × R1 / (R_поплавка + R1)
```

| Состояние | R_поплавка | Vadc | ADC (12-bit) |
|-----------|-----------|------|--------------|
| **Полный бак** | 0 Ом | 3.3 × 100 / (0+100) = **3.30В** | **4095** |
| **Полупустой** | 54 Ом | 3.3 × 100 / (54+100) = **2.14В** | **2655** |
| **Пустой бак** | 107 Ом | 3.3 × 100 / (107+100) = **1.59В** | **1980** |

**Диапазон:** 1.59В → 3.30В (Δ = 1.71В)

### Разрешение ADC

ESP32 ADC: 12 бит = 4096 уровней (0-4095)

```
Разрешение = 3.3В / 4095 = 0.8 мВ/уровень
Рабочий диапазон = 4095 - 1980 = 2115 уровней
Точность = 100% / 2115 = 0.047% (отлично!)
```

## Подключение к Toyota RAV4

### Шаг 1: Найти разъём бензонасоса

**Расположение:** Под задним сиденьем (лючок доступа)

**Разъём:** 6-пиновый (Toyota тип E)

### Шаг 2: Определить пины поплавка

| Пин | Назначение | Цвет провода (типично) |
|-----|-----------|------------------------|
| **FUA** | Поплавок основной | Белый/Чёрный |
| **FU** | Поплавок дополнительный | Белый/Чёрный (другой) |
| **E2** | Масса (GND) | Коричневый |

⚠️ **Важно:** Проверить мультиметром!

### Шаг 3: Проверка поплавка мультиметром

1. Отсоединить разъём от ЭБУ
2. Замерить сопротивление между пинами поплавка:
   - Полный бак: ~0-5 Ом
   - Средний уровень: ~50-60 Ом
   - Пустой бак: ~105-110 Ом

## Калибровка (обязательно!)

### Шаг 1: Полный бак

1. Заправить **до полного** (до отсечки)
2. Включить зажигание (не запуская двигатель)
3. Замерить напряжение на GPIO34 мультиметром:
   ```
   Vadc_full = ??? В (записать!)
   ```

### Шаг 2: Пустой бак

1. Проехать до загорания лампочки (остаток ~5-7 л)
2. Замерить напряжение на GPIO34:
   ```
   Vadc_empty = ??? В (записать!)
   ```

### Шаг 3: Обновить код

В файле `real_engine.cpp`:

```cpp
static float    fuelAdcFull     = 3.30f;   // ← Заменить на замеренное!
static float    fuelAdcEmpty    = 1.59f;   // ← Заменить на замеренное!
```

## Компоненты

| Компонент | Номинал | Кол-во | Назначение |
|-----------|---------|--------|-----------|
| **R1** | 100 Ом, 1% | 1 | Фиксированный резистор делителя |
| **C1** | 100 нФ (опционально) | 1 | Фильтр помех (параллельно ADC-GND) |

### Требования к резисторам

- **Точность:** 1% (цветовая маркировка: коричневый-чёрный-чёрный-золотой)
- **Мощность:** 0.125 Вт (стандартные 0805/1206)
- **Температурный коэффициент:** 100 ppm/°C (металлоплёночные)

---

## Фильтрация помех

Автомобиль — среда с сильными электромагнитными помехами. Для стабильных показаний:

### Аппаратная фильтрация

```
         ┌───[0-107 Ом]───┬─── GPIO34
    3.3V ┤                │
         │           [100 нФ]
         │                │
         └───[100 Ом]─────┴─── GND
```

**RC-фильтр:**
- R = 100 Ом (эквивалент)
- C = 100 нФ
- Частота среза: f = 1 / (2π × R × C) = **16 кГц**

### Программная фильтрация

Реализовано в `readFuelAdc()`:

```cpp
// Экспоненциальное сглаживание: α = 0.25
adcSmooth = adcSmooth × 0.75 + volts × 0.25
```

**Время реакции:** ~4 измерения (100 мс)  
**Подавление шума:** ~10×

## Troubleshooting

### Проблема: ADC показывает 0В или 3.3В постоянно

**Причины:**
- Обрыв провода поплавка
- Короткое замыкание
- Неправильный пин GPIO

**Решение:**
1. Проверить мультиметром сопротивление поплавка (должно быть 0-107 Ом)
2. Проверить целостность проводов
3. Убедиться, что GPIO34 (только вход!)

### Проблема: Показания плавают ±10%

**Причины:**
- Топливо плещется в баке
- Наклон автомобиля
- Помехи от бортсети

**Решение:**
1. Увеличить сглаживание в `readFuelAdc()` (α = 0.1 вместо 0.25)
2. Добавить RC-фильтр (100 нФ)
3. Использовать **оба** поплавка (уже последовательно — компенсация наклона)

### Проблема: Полный бак = 0%

**Причины:**
- Перепутаны `fuelAdcFull` и `fuelAdcEmpty`
- Неправильная формула

**Решение:**
- Полный бак = **больше** Вольт (сопротивление поплавка ≈ 0)
- Пустой бак = **меньше** Вольт (сопротивление поплавка ≈ 107 Ом)
- Формула: `pct = (Vadc - fuelAdcEmpty) / (fuelAdcFull - fuelAdcEmpty) × 100%`

## Пример кода (тест вне авто)

```cpp
void setup() {
    Serial.begin(115200);
    analogReadResolution(12);
    pinMode(34, INPUT);
}

void loop() {
    int raw = analogRead(34);
    float volts = raw * (3.3 / 4095.0);
    
    // Расчёт уровня топлива (подставить свои значения!)
    float full = 3.30;   // Вольт (полный бак)
    float empty = 1.59;  // Вольт (пустой бак)
    float pct = (volts - empty) / (full - empty) * 100.0;
    pct = constrain(pct, 0, 100);
    
    Serial.printf("ADC: %d, V: %.3f, Fuel: %.1f%%\n", raw, volts, pct);
    delay(500);
}
```

**Тест с резисторами:**
```
R_поплавка = 0 Ом    → Fuel: 100%
R_поплавка = 54 Ом   → Fuel: 50%
R_поплавка = 107 Ом  → Fuel: 0%
```
